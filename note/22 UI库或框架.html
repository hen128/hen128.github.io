# 更改主题色

参考官网：https://element.eleme.cn/#/zh-CN/component/custom-theme

新建文件 element-variables.scss

mary: teal;

/* 改变 icon 字体路径变量，必需 */
$--font-path: '~element-ui/lib/theme-chalk/fonts';

@import "~element-ui/packages/theme-chalk/src/index";
